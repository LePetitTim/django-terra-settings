Example of use
==============

- Set appliance settings

::

    TERRA_APPLIANCE_SETTINGS = {
        "title": "APP TITLE",
        # temp theme settings
        "theme": {
            "logo": {
                # base64
                "src": "data:image/png;##########################################",
                "alt": "ALT LOGO TEXT"
            },
        },
        # favicon settings
        # base64
        "favicon": "data:image/png;##########################################",
        # "landing_module": "CRUD",  optional, defined a landing module instead of select module landing page
    }
